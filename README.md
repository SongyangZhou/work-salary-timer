# 💰 Work Salary Timer

> Watch your earnings grow in real time — the emotional support app for workers everywhere.

**[🚀 Live Demo](https://songyangzhou.github.io/work-salary-timer/)**

---

## What is this?

A single-file web app that tracks your salary in real time, down to the cent. Enter your monthly salary and work hours, and watch the counter tick up every second you're on the clock.

Because sometimes the only thing getting you through a pointless meeting is knowing you just made $4.37 sitting there.

## Features

- **⚡ Real-time earnings counter** — updates every 100ms with 2 decimal places
- **🪙 Milestone coin rain** — hit your reward target and gold coins pour from the sky (with sound)
- **🏆 Growth level system** — level up from *Rookie Grinder* to *Financial Freedom*
- **🛒 Purchasing power panel** — see how many coffees/burgers/movie tickets you've earned today
- **📅 Payday countdown** — exact hours until your next paycheck
- **💼 Overtime mode** — tracks post-hours time at 1.5×
- **📊 Fun stats** — earnings per second, per minute, annual estimate
- **📢 Office event ticker** — rotating workplace humor to get you through the day

## Usage

No install, no build step, no server needed.

1. Open `index.html` in any browser
2. Enter your monthly salary, work hours, and payday date
3. Watch the money flow

Settings are saved to `localStorage` — your data never leaves your device.

## Tech

Pure HTML + CSS + JavaScript. Single file, zero dependencies, zero tracking.

Web Audio API synthesizes the coin sound effect — no external audio files.
